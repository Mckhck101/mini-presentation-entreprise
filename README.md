# Mini-site de Présentation d'Entreprise

## Structure
- `index.html`, `services.html`, `contact.html`
- `styles/` : thème CSS
- `scripts/` : interactions JS

## Branches
- `main` : version finale
- `dev` : intégration
- `feature/*` : fonctionnalités
- `test/ui-review` : tests UI/UX

## Développement
- Git Flow utilisé pour l'organisation du projet.
